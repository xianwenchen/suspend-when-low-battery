#+title: swlb
#+author: Xianwen Chen

swlb is a shell script that calls zzz when battery level is low.
swlb means suspend when low battery.

The script is written for Void Linux.

The source code is written in the literate programming style.
See [[file:DOCUMENTATION/swlb-design.org]].

The script can be adapted to other Linux distributions and to other UNIX-like systems.
